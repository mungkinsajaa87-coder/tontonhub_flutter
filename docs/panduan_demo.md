# Panduan Demo TontonHub

## Urutan Demo

1. Buka aplikasi di Chrome/web.
2. Tampilkan halaman login.
3. Register akun Subscriber.
4. Login sebagai Subscriber.
5. Tampilkan dashboard Subscriber.
6. Edit profil dan upload foto profil.
7. Buka halaman info.
8. Buka halaman video.
9. Gunakan fitur search video.
10. Buka detail video.
11. Putar video dengan video player.
12. Tambahkan komentar.
13. Klik Subscribe Channel.
14. Klik Simpan Riwayat Tonton.
15. Tampilkan halaman riwayat.
16. Logout.
17. Login/register sebagai Creator menggunakan kode CREATOR123.
18. Tampilkan dashboard Creator.
19. Tambah info.
20. Tambah video dan upload thumbnail.
21. Tampilkan data subscribe.
22. Buka aplikasi di Android/mobile sebagai bukti multiplatform.
23. Tampilkan Firebase Authentication, Firestore, dan Storage.
24. Tampilkan hasil `flutter test`.

## Kalimat Pembuka Demo

Assalamualaikum Pak/Bu, saya akan mendemokan aplikasi TontonHub. TontonHub adalah aplikasi Video Streaming Platform berbasis Flutter dan Firebase. Aplikasi ini memiliki dua role pengguna, yaitu Creator dan Subscriber. Subscriber dapat menonton video, komentar, subscribe channel, dan melihat riwayat tontonan. Creator dapat mengelola info, menambahkan video, mengupload thumbnail, dan melihat data subscribe.
